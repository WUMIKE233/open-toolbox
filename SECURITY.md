# Security Policy

## Supported Versions

Security fixes are considered for the latest version on the default branch.

## Reporting a Vulnerability

Please report security issues privately through GitHub's private vulnerability reporting if it is enabled for the repository. If it is not enabled, open a minimal issue that asks maintainers for a private reporting channel and avoid sharing exploit details publicly.

## Scope

The tools in this repository are local CLIs. They do not make network requests unless a future feature explicitly documents that behavior.
